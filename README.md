# Assignment 0: Introduction to Google Colab & GitHub

Welcome to the YSE Data Science Certificate Program! This assignment will help you get set up with Google Colaboratory and GitHub, the two tools we'll use throughout the program.

## Open the Assignment Notebook

Click the button below to open the assignment notebook in Google Colab:

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/yse-eds-cert/assignment-0/blob/main/assignment_0.ipynb)

> **Important:** This button opens the starter notebook from the template. Once it opens in Colab, immediately go to **File → Save a copy in GitHub** and save it to **your own assignment repository** (e.g., `yse-eds-cert/assignment-0-yourname`). All your work should be saved to your personal repo.

## Instructions

1. Click the **Open in Colab** button above.
2. In Colab, go to **File → Save a copy in GitHub** and select your personal assignment repo.
3. Complete the exercises in the notebook using the **R kernel**.
4. When finished, save your work back to GitHub via **File → Save a copy in GitHub**.
5. Your submission is complete when your completed notebook appears in your GitHub repo.
