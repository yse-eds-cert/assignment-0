# Assignment 0: Introduction to Google Colab & GitHub

Welcome to the YSE Environmental Data Science Certificate Program! This assignment will walk you through the tools we'll use throughout the program: Google Colaboratory (Colab) and GitHub.

## How to Open the Assignment Notebook

> ⚠️ **Do not use an Open in Colab button from this README.** Instead, follow the steps below so that Colab knows to save your work to **your own** assignment repo.

1. **Accept the assignment** via the invitation link — GitHub Classroom will create your personal repo (e.g., `yse-eds-cert/assignment-0-yourname`).
2. **Go to your personal repo** on GitHub (e.g., `https://github.com/yse-eds-cert/assignment-0-yourname`).
3. **Click on `assignment_0.ipynb`** — GitHub will show an **"Open in Colab"** button at the top of the file. Click that.
4. In Colab, go to **File → Save a copy in Drive** — this creates an autosaving working copy in your Google Drive. Do all your work here.
5. Then go to **File → Save a copy in GitHub** → select your personal assignment repo (`yse-eds-cert/assignment-0-yourname`) to link it right away.
6. Complete the exercises in the notebook using the **R kernel**.
7. When finished, submit by going to **File → Save a copy in GitHub** again to push your final work to your GitHub repo.

> **Why save to Drive first?** Colab autosaves to Google Drive but does **not** autosave to GitHub. Saving to Drive protects your work-in-progress. Saving to GitHub is the manual submission step.
