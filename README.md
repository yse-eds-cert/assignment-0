# Assignment 0: Introduction to Google Colab & GitHub

Welcome to ENV 617! This assignment will walk you through the tools we'll use throughout the course: Google Colaboratory (Colab) and GitHub.

## Open the Assignment Notebook

Click the button below to open the assignment notebook in Google Colab:

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/yse-eds-cert/assignment-0/blob/main/assignment_0.ipynb)

## Instructions

1. Click the **Open in Colab** button above.
2. **File → Save a copy in Drive** — this creates an autosaving working copy in your Google Drive. Do all your work here.
3. **File → Save a copy in GitHub** → select your personal assignment repository (`yse-eds-cert/assignment-0-yourname`). Do this right away so your repo is linked.
4. Complete the exercises in the notebook using the **R kernel**.
5. When finished, submit by going to **File → Save a copy in GitHub** again to push your final work to your GitHub repo.

> **Why save to Drive first?** Colab autosaves to Google Drive but does **not** autosave to GitHub. Saving to Drive protects your work-in-progress. Saving to GitHub is the manual submission step.
